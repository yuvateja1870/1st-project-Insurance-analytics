# 1st-project-Insurance-analytics

# PROJECT OVERVIEW
- This project analyzes insurance business performance using SQL and Power BI/Tableau. The goal of this project is to track business targets, performance metrics, and sales insights.

# DATASET DESCRIPTION
The project uses insurance business data including:
- Target Data – Business planning targets
- Opportunity Data – Sales pipeline information
- Meeting Data – Client interaction records
- Invoice Data – Billing information
- Brokerage & Fees Data – Revenue related data

# TOOLS USED
- Excel – Data Cleaning
- SQL – Data Analysis
- Power BI/Tableau – Dashboard Visualization
